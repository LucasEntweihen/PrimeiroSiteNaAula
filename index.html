Passos para Hospedar um Projeto no Github Pages

  1. Crie um repositório no Github com o mesmo nome do seu projeto.
  2. No terminal, inicialize o repositório local git init
  3. Altere o nome da branch principal para main.
  4. Adicione os arquivos com got add . o faça o commit com git commit -m "first commit".
  5. Conect ao repositório remoto com:
    git remote add origin https://github.com/SEU_USUARIO/HOME_DO_REPOSITORIO.git
  6. Envie os arquivos com git push -u origin main.
  7. No Github, vá em Settings > Pages.
  8. Em "Branch", selecione main e a pasta / (root) cilque em Save.
  9. Após alguns segundos, seu site estará disponível no link exibido em "Pages".
